# OpenPrints Code of Conduct

OpenPrints is an open-source, bootstrapped project. We want our community to be a friendly, safe, and welcoming place for everyone interested in censorship-resistant 3D-printing tools.

This Code of Conduct applies to all project spaces, including:

- Git repositories, issues, and pull requests
- Discussion forums, chat channels, and social media used for project coordination
- Any other project-related communication, online or offline

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and considerate in your communication
- Giving and gracefully accepting constructive feedback
- Focusing on what is best for the project and its users
- Showing empathy and support for other community members
- Using inclusive language and assuming good faith

Examples of unacceptable behavior include:

- Harassment, insults, or personal attacks
- Discriminatory language or behavior (including but not limited to racism, sexism, homophobia, transphobia, ableism, or religious intolerance)
- Publishing others’ private information (doxxing) without explicit permission
- Sustained disruption of conversations, issues, or project work
- Any conduct that could reasonably be considered inappropriate in a professional setting

## Responsibilities and Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may take appropriate and fair corrective action in response to any behavior they consider unacceptable, including:

- Requesting a change in behavior
- Removing comments, code, or other contributions that violate this policy
- Temporarily or permanently banning individuals from project spaces

Maintainers must communicate reasons for major enforcement actions when appropriate, while respecting the privacy of those involved.

## Reporting Concerns

If you experience or witness behavior that violates this Code of Conduct, or have any other concerns, please contact the maintainers at:

**`openprints@pm.me`**

Please include:

- A description of what happened
- Where and when it occurred (links, screenshots, timestamps if available)
- Any additional context you think might be helpful

We will review all reports, keep them as confidential as reasonably possible, and respond in good faith.

## Scope

This Code of Conduct applies:

- Within all official OpenPrints project spaces, and
- When someone is officially representing the project in public spaces (for example, using project-branded accounts, or speaking on behalf of OpenPrints).
